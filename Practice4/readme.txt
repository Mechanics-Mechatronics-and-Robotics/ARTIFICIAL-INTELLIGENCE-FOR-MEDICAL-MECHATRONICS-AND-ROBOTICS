Практика использования глубокой сверточной нейронной сети AlexNet в пакете MATLAB. Рассматривается задача распознавания грустных и веселых смайликов на изображениях размером 30х30х3. 
Предварительно необходимо установить аддоны Deep Learning Toolbox, Deep Learning Toolbox Model for AlexNet Network с официального сайта.
Видео инструкция выполнения задания на канале Mechanics,Mechatronics and Robotics платформы YouTube: https://youtu.be/y2e7TxyuHhI
